NBUAssetsGroup Class Reference

Inherits from NSObject
Declared in NBUAssetsGroup.h

Overview

Wrapper to ease acces to an ALAssetsGroup assets group.

  • Unlike ALAssetsGroup objects, NBUAssetsGroup is always valid.
  • Observes ALAssetsLibraryChangedNotification to reload its associated
    ALAssetsGroup if needed.
  • Lazily loads posterImage and assets' count only when needed.
  • Leverages iOS4 compatibility by creating a NSURL:
    assets-library://group/?id=<ALAssetsGroupPropertyPersistentID>.

%note%
Note: You usually retrieve assets groups using NBUAssetTypesLibrary methods.

assetsWithTypes:atIndexes:reverseOrder:incrementalLoadSize:resultBlock:

Returns the assets matching an ALAssetsFilter.

- (void)assetsWithTypes:(NBUAssetType)types atIndexes:(NSIndexSet *)indexSet reverseOrder:(BOOL)reverseOrder incrementalLoadSize:(NSUInteger)loadSize resultBlock:(NBUAssetsResultBlock)resultBlock

Parameters

types

The desired type mask of assets to retrieve.

indexSet

The index of the desired assets. Pass nil to get all the assets.

reverseOrder

Set to YES to get newer to older assets.

loadSize

If different to zero the resultBlock will be called after loading another
loadSize number of assets.

resultBlock

The block to be called asynchronously with the results.
